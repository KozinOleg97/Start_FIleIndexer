Программа для индексирования файлов техпроцессов с формированием отчёта в формате .xls
ТРЕБУЕТ .NET Framework 4.0

Исполняемый файл Start_FIleIndexer.exe, является консольным приложением, ничего не выводит, и не требует ввода данных. 
Для функционирования программы необходимы все файлы в папке с исполняемым файлом.


Кофигурация программы производится в файле Start_FIleIndexer.exe.config (открывать как тестовый файл и редактировать). 
В файле конфигурации ОПИСАНЫ значния полей, значения заносятся в виде value="КАКОЕ_ТО_ЗНАЧЕНИЕ".

Для работы нужно указать 3 папки:
	- Папка в которой хронятся ИСХОДНЫЕ(не имеющих индекса) файлы
	- Папка в которой будут находится проиндексированные КОНЕЧНЫЙ файлы
	- Папка для хранения отчёта в формате excele


Программа НЕ расчитана на внесение ручных изменений в excele файл отчёта!

Удаление ИСХОДНЫХ файлов техпроцессов не влияет на работоспособность.
* Программа учитывает только ИСХОДНЫЕ файлы не занесённые в отчёт и производит формирование КОНЕЧНЫХ файлов и вносит их в отчёт.

Программа расчитана на периодическое выполнение (с помощью планировщика событий).
